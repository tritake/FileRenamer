=====================================
  サウンドボード - 使い方ガイド
=====================================

ボイスチャット中にホットキーでMP3を再生できるアプリです。
自分の声と一緒にMP3が相手に聞こえます。


【 事前準備（初回のみ） 】
--------------------------------------

1. VB-Audio Virtual Cable をインストール
   https://vb-audio.com/Cable/
   → VBCABLE_Setup_x64.exe を「管理者として実行」→「Install Driver」
   → インストール後、PCを再起動

2. ボイスチャットアプリ（Discord等）の設定を変更
   → 設定 → 音声・ビデオ → 入力デバイス
   → 「CABLE Output (VB-Audio Virtual Cable)」を選択


【 アプリの使い方 】
--------------------------------------

1. Soundboard.exe をダブルクリックで起動

2. デバイスを設定
   - 入力 (マイク)     : 自分の物理マイクを選択
   - 出力 (仮想ケーブル) : 「CABLE Input (VB-Audio Virtual Cable)」を選択
   - モニター (自分用)   : 自分のスピーカー/イヤホンを選択（任意）

3. サウンドを追加
   - 「追加」ボタン → MP3ファイルを選択（複数OK）

4. ホットキーを割り当て
   - サウンド一覧からMP3をクリックして選択
   - 「キー設定」ボタンを押す
   - 割り当てたいキーまたはマウスボタンを押す
     （例: F1, F2, マウスのサイドボタン など）

5. 「開始」ボタンを押す
   → 「動作中 - マイク通過」と表示されればOK！


【 操作方法 】
--------------------------------------

・何もしない時     → 普通にマイクの声が相手に聞こえる
・ホットキーを押す → MP3が再生される（声もミックスされる）
・別のキーを押す   → 今の音を中断して別の音に切り替え
・「再生停止」     → MP3の再生だけ止める
・「停止」         → アプリ全体を停止（マイク通過も止まる）


【 音量調整 】
--------------------------------------

・マイク   : 相手に聞こえる自分の声の大きさ
・サウンド : 相手に聞こえるMP3の大きさ
・モニター : 自分に聞こえるMP3の大きさ
  ※ 0〜200% まで調整可能


【 設定の保存 】
--------------------------------------

デバイス・サウンド・音量の設定は自動で保存されます。
次回起動時にそのまま復元されます。


【 注意事項 】
--------------------------------------

・VB-Audio Virtual Cable がインストールされていないと動きません
・アプリ起動中はマイク音声がアプリ経由で送られます
・アプリを閉じると通常のマイク入力に戻ります
  （Discordの入力デバイスを元に戻すか、アプリを再起動してください）
